#Problem 1

Reverse Words in a String

Problem Statement:

Given an input string, reverse the words of the string. A word is defined as a sequence of non-space characters. The words in the string will be separated by one or more spaces. Your reversed string should not contain leading or trailing spaces.

Example:

Input:

"The sky is blue"

Output:

"blue is sky The"

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#Solution in Python


class Solution:
    def reverseWord(self, s: str) -> str:
        result, i, n = '', 0, len(s)
        while i < n:
            while i < n and s[i] == ' ':
            i += 1
            j = i + 1
            while j < n and s[j] != ' ':
            j += 1
            sub = s[i:j]
            if len(result) == 0:
                result = sub
            else:
                result = sub+" "+result
            i = j + 1
        return result
