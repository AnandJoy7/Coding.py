#Problem 2

Longest Palindromic Substring
Given a string, find the longest palindromic contiguous substring. If there are more than one with the maximum length, return any one.

A palindrome is a word, phrase, number, or other sequences of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization).

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
#Solution 

class Solution:
    def longestPalindrome(self, s): 
        def check(l,r):
            while(l >= 0 and r < len(s) and s[l] == s[r]):
                l -= 1
                r += 1
            return s[l+1:r]
        result=""
        for i in range(len(s)):
            sub1 = check(i ,i)
            if len(sub1) > len(result):
                result = sub1
            sub2 = check(i, i+1)
            if len(sub2) > len(result):
                result = sub2
        return result
